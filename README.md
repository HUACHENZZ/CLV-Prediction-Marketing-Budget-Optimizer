## 📖 Project Overview

This project builds an end-to-end analytics solution that:
1. **Predicts Customer Lifetime Value (CLV)** using machine learning
2. **Identifies high-value customer segments** for targeted retention
3. **Optimizes marketing budget allocation** using ROI simulation

## 📊 Key Features

### 1. CLV Prediction Model
- **Algorithm**: Random Forest Regressor
- **Performance**: R² = 0.93, RMSE = $125.97
- **Interpretability**: SHAP values for feature importance

### 2. Customer Segmentation
- **Segments**: Low, Medium, High, Champion
- **Method**: Quantile-based CLV segmentation
- **Insight**: Top 25% (Champions) contribute 60%+ of revenue

### 3. ROI Simulator
- **What-If Parameters**: Budget, Cost Per User, Success Rate
- **Output**: Net ROI, Recommendation (Go/No-Go)
- **Use Case**: Marketing budget decision support
